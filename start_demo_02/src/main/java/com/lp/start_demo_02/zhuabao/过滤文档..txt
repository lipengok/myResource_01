# 根据协议（protocol）过滤
   tcp，只显示TCP协议的数据包列表
   http，只查看HTTP协议的数据包列表
   icmp，只显示ICMP协议的数据包列表

# 根据ip地址过滤
   ip.src ==192.168.1.104 显示源地址为192.168.1.104的数据包列表
   ip.dst==192.168.1.104, 显示目标地址为192.168.1.104的数据包列表
   ip.addr == 192.168.1.104 显示源IP地址或目标IP地址为192.168.1.104的数据包列表

# 分析抓包文件
   SYN表示建立连接
   FIN表示关闭连接
   ACK表示响应
   PSH表示有DATA数据传输
   RST表示连接重置

# 情景分析（TCP连接）
   [SYN]：请求建立连接
   [SYN,ACK]：建立连接成功
   [PSH,ACK]：发送数据
   [ACK]：确认连接是否断开
   [RST,ACK]：断开连接
