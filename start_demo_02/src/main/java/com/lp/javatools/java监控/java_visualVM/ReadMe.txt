总结
1，java visualVM如何安装
Java se7之后jdk（Java工具包）自带。

2，java visualVM如何启动
win+R，之后输入jvisualvm

3，java visualVM如何使用
首先，进入之后，添加远程主机
接着，添加主机之后，右键主机添加jmx（端口号）
最后，双击主机就可以进行监控
